Landing page
for game arena: Force drop zone

Role

Web designer

Contribution

UX research, competitor analysis, information architecture, user flow design, UI design in Figma, and full-stack development (frontend + Node.js backend with Telegram Bot API integration).

(1)

Have to switch

Users don’t understand how skin giveaways work or whether they can trust them. Information is scattered, unclear, and there is no transparent history of past drops, which creates doubt and lowers participation.

(2)

Solution

A gaming landing page that clearly explains how daily skin drops work, builds trust through transparent drop history, and allows users to easily join via a simple booking form connected to Telegram.

(3)

Design principles

A dark, game-inspired UI aligned with CS2 aesthetics, creating an immediate connection with the target audience.

Clear information hierarchy that explains the drop mechanism step-by-step, reducing cognitive load for new users.

Trust-driven design with visible history of past giveaways to increase credibility and engagement.

(4)

Position

A transparent and engaging space where users instantly understand how drops work and can confidently participate without confusion or doubt.

(5)

Drop history

A dedicated section showcasing past skin giveaways, helping users verify authenticity and build trust in the system. It reinforces transparency and encourages participation by showing real outcomes.

(7)

Participation Flow

A simple booking form integrated with Telegram Bot API that allows users to instantly join daily drops. The streamlined flow reduces friction and makes participation fast and accessible.
